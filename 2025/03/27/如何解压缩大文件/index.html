如何解压大文件

1. 常见解压方式及优化

使用 7z 解压

  • -mmt=48：使用 48 线程（根据 CPU 核心数调整）。
  • -ooutput_dir：指定输出目录。

续解压（跳过已解压文件） 7z x file.zip -ooutput_dir -aos -mmt=48

使用 unzip

  • -d output_dir：解压到指定目录。

继续上次的解压

结论

  • 使用 7z 和 unzip -n 避免重复解压。
